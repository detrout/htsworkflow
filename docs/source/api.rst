Runfolder Processing
====================

Runfolder
---------

The PipelineRun class is designed to combine information
from the following importers.

* Image Analysis (one of the following)

  * :class:`Firecrest`
  * :class:`IPAR`

* BaseCaller

  * :class:`Bustard`

* Sequence Alignment

  * :class:`Gerald`
